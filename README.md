# 🧵 Vishal Textiles Group - Landing Page

A modern, high-performance Single-Page Application (SPA) designed to serve as the official advertising and contact portal for Vishal Textiles Group. 

This project aims to bridge the gap between traditional textile manufacturing and a sleek, modern web presence, utilizing smooth scroll mechanics, high-end UI components, and SEO-optimized rendering.

---

## 🚀 Tech Stack

*   **Framework:** [Next.js](https://nextjs.org/) (React)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Animations:** [Framer Motion](https://www.framer.com/motion/)
*   **Form Handling:** To Be Determined (e.g., Node.js backend, EmailJS, or Resend)
*   **Deployment:** Vercel (Recommended)

---

## 🎨 Design System & UI/UX

The design language pulls from the official Vishal Textiles logo, utilizing a tech-forward aesthetic with crisp contrasts. 

**UI Aesthetic:** 
Modern, clean layouts featuring subtle **glassmorphism** effects (frosted glass overlays with light borders) over dark backgrounds to give the site a polished, premium feel.

**Color Palette:**
*   ⬛ **Deep Navy (Primary Dark):** `#060B1E` *(Hero background, footer, primary text on light sections)*
*   🟦 **Vibrant Cyan (Accent):** `#2BD3FF` *(CTA buttons, active links, icons, hover states)*
*   ⬜ **Pure White (Background):** `#FFFFFF` *(Cards, standard backgrounds, text on dark sections)*
*   🪙 **Silver/Grey (Secondary):** `#C0C5CE` *(Subtle borders, secondary text)*
*   🌫️ **Slate Light (Divider):** `#F3F4F6` *(Alternating section backgrounds)*

---

## 🗺️ Project Architecture (Single-Page Flow)

1.  **Hero Section:** High-impact logo display, deep navy background, bold value proposition, and a primary CTA.
2.  **About Us:** Company history and mission statement.
3.  **Our Expertise/Products:** Grid or carousel showcasing textile categories (cottons, synthetics, etc.).
4.  **Quality Assurance:** Highlighting manufacturing tech and sustainability.
5.  **Contact & Footer:** Contact form, physical address, map embed, and social links.
